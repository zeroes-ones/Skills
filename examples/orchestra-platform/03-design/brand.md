# Brand Guidelines — Orchestra Platform

**Skill:** brand-guidelines
**Input:** Product name, target audience, competitive landscape

## Brand Architecture

**Master brand:** Orchestra (the IDP platform)
**Sub-brands:** Orchestra Plugin [Name] (partner plugins), Orchestra CLI (developer tool)

Single brand with endorsed sub-brands. All derivative names include "Orchestra" for recognition. No standalone plugin brands — they belong to the platform ecosystem.

## Identity System

### Logo

**Symbol:** Interlocking circles forming an abstract "O" — representing services connecting in a platform
**Wordmark:** "Orchestra" in Inter Bold, tracking: -0.5px
**Clear space:** 1x logo height on all sides
**Minimum size:** 32px wide for digital, 0.5in for print

### Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Midnight | #1A1A2E | Primary background, headers |
| Indigo | #16213E | Secondary background, cards |
| Teal | #0F3460 | Interactive elements, links, CTAs |
| Coral | #E94560 | Accents, notifications, errors |
| Platinum | #F5F5F5 | Text on dark, light backgrounds |
| Mint | #53D8B2 | Success states, confirmation |

**Accessibility:** All combinations verified at WCAG AA (4.5:1 minimum for normal text, 3:1 for large text). Midnight + Platinum: 15.2:1 (AAA). Indigo + Mint: 6.8:1 (AA). Coral on Midnight: 4.9:1 (AA).

### Typography

**Headings:** Inter (weights: Bold 700, SemiBold 600)
**Body:** Inter Regular 400, 16px base, 1.6 line-height
**Code:** JetBrains Mono (weights: Regular 400, Medium 500)
**Fallback stack:** Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif

Type scale: 12px (caption), 14px (small), 16px (body), 20px (h4), 24px (h3), 32px (h2), 48px (h1)

### Iconography

**Style:** Outline, 1.5px stroke, rounded corners (2px radius)
**Grid:** 24x24px
**Library:** Phosphor Icons (MIT licensed)
**Custom icons:** Service, Template, Plugin, Catalog — custom for core concepts

### Imagery

**Style:** Dark backgrounds with glowing teal/coral accent lines. Abstract geometric patterns suggesting connectivity. No stock photos of people — use abstract tech imagery.
**Illustrations:** Isometric 3D service diagrams, consistent lighting (top-left light source), teal + coral palette.
**Screenshots:** Product UI with Midnight background, browser frame, subtle shadow.

## Brand Governance

- All brand assets in Figma library (single source of truth)
- Brand reviewed quarterly against competitive landscape
- Plugin partners get co-branding guidelines (Orchestra Plugin badge + partner logo)
- No unauthorized color or logo variations — PR review required for brand asset changes
