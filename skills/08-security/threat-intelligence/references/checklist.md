# Production Checklist

- [ ] CTI platform (MISP/OpenCTI) deployed and operational with automated feed ingestion
- [ ] Priority Intelligence Requirements (PIRs) documented and approved by stakeholders
- [ ] Collection plan maps each PIR to specific intelligence sources
- [ ] All intelligence products published in STIX 2.1 format
- [ ] IOC lifecycle automated: ingestion → enrichment → expiration (decay curve)
- [ ] Threat actor dossiers maintained for all relevant APT groups and ransomware families
- [ ] MITRE ATT&CK mapping completed for all tracked threat actors
- [ ] Intelligence disseminated to SOC (tactical), detection engineering (operational), and leadership (strategic) on defined cadence
- [ ] Threat hunts conducted at least monthly, hypothesis-driven, with documented findings
- [ ] Intelligence sharing agreements active with relevant ISACs/ISAOs
- [ ] Dark web monitoring capability operational
- [ ] Intelligence feedback loop active — consumers rate product relevance and timeliness quarterly
