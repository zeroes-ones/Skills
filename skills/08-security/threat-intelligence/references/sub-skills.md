# Sub-Skills

| Sub-Skill | Description | When to Route |
|---|---|---|
| **Threat Actor Profiling** | Deep-dive attribution analysis, APT group dossier maintenance, diamond model mapping, infrastructure tracking | When attributing an intrusion to a specific threat actor or building actor dossiers |
| **IOC Lifecycle Management** | IOC ingestion, enrichment, confidence scoring, expiration/decay, false positive triage | When building automated IOC pipelines or tuning IOC quality |
| **Strategic Intelligence** | Executive threat landscape briefings, industry trend analysis, geopolitical risk assessment, board-level reporting | When preparing intelligence for non-technical stakeholders or strategic planning |
| **Threat Hunting** | Hypothesis-driven hunt methodology, TTP-based detection engineering, hunt documentation and metrics | When designing or executing proactive threat hunting operations |
| **Intelligence Sharing** | ISAC/ISAO participation, STIX/TAXII federation, trust group management, disclosure controls | When establishing intelligence sharing relationships or automated exchange pipelines |
| **Dark Web Monitoring** | Ransomware leak site tracking, forum monitoring, credential exposure detection, marketplace surveillance | When monitoring for organizational exposure on dark web and criminal forums |
