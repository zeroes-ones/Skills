# Calibration — How to Know Your Level

| Level | You Know You're At This Level When... |
|---|---|
| **L1 (Novice)** | You can generate an SBOM with `syft` or `cyclonedx-cli` and run `cosign verify` on a container image. You understand what SLSA is but haven't implemented it. |
| **L2 (Competent)** | You've implemented SLSA Level 2 in a CI pipeline, configured Dependabot with auto-merge rules, and set up artifact signing with OIDC-based keyless signing. |
| **L3 (Proficient)** | You've designed an organization-wide supply chain security program including SBOM lifecycle management, vendor attestation requirements, and provenance verification gates. |
| **L4 (Expert)** | You contribute to supply chain security standards (SLSA, in-toto, Sigstore), design custom attestation predicates, and advise regulators on supply chain security policy. |
| **L5 (Master)** | You shape the industry — your supply chain security framework is adopted by multiple organizations, you've published novel attack research, and you contribute to OWASP Top 10 supply chain guidance. |
