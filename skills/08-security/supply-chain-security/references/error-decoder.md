# Error Decoder

| Error Message | What It Really Means | Fix |
|---|---|---|
| `cosign: error: no matching signatures` | The image has no signature or the signing identity doesn't match your expected identity | Verify the image digest, check that the expected OIDC issuer and subject match, run `cosign triangulate` to find the signature |
| `slsa-verifier: untrusted builder` | The provenance was generated by a builder not in your trusted builder list | Add the builder ID to your trusted list, or investigate why an unexpected builder generated the provenance |
| `npm ERR! 403 Forbidden` on private package | Registry is resolving to the public registry for a scoped package | Check `.npmrc` for correct `@scope:registry` configuration; verify the auth token has package read permissions |
| `in-toto: layout verification failed` | A step in the supply chain layout didn't produce expected artifacts or the functionary key doesn't match | Check each step's expected materials/products, verify functionary public keys, re-run the layout with `--verbose` |
| `syft: unsupported format` | The SBOM format requested isn't supported by the installed syft version | Update syft or use CycloneDX/SPDX which are universally supported |
