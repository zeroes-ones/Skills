# Socratic Question Patterns for Design Exploration

## Pattern Catalog

### Clarifying Questions
- "What do you mean by [term]? Can you define it precisely?"
- "When you say [X], is that an observation or an assumption?"
- "Can you give me a concrete example of [scenario]?"

### Probing Questions
- "What would happen if we DIDN'T do this?"
- "Who benefits most from this decision, and who loses?"
- "What would the 'do nothing' approach cost us?"

### Challenging Questions
- "What evidence would change your mind about this approach?"
- "If the opposite were true, what would you choose instead?"
- "What assumption are you making that, if false, invalidates this entire design?"

### Perspective-Shifting Questions
- "How would a competitor approach this problem?"
- "How would a junior engineer with no context solve this differently?"
- "If budget were unlimited, what changes? If we had $0 budget, what changes?"

### Constraint-Surfacing Questions
- "What can't we change about this situation?"
- "What problem did this existing constraint originally solve?"
- "What would we lose if we removed this limitation?"

## Sequencing Guide
Use clarifying first, probing second, challenging third. Constraint-surfacing before decision-making. Perspective-shifting when stuck.
