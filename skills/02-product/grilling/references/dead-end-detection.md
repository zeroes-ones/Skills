# Dead-End Detection: When Branches Cannot Be Resolved

## Dead-End Taxonomy

### Type 1: Missing Information
The information needed to decide does not exist yet.
- Response: Mark DEFERRED. Specify what information is needed, who will gather it, and by when.
- Escalation: If information is unknowable (e.g., market behavior in 5 years), treat as ASSUMED with contingency.

### Type 2: Conflicting Constraints
Two constraints demand opposite decisions and neither can be relaxed.
- Response: Ask "Which constraint would you relax if you had to? What's the least-bad option?"
- Escalation: If no constraint can be relaxed, escalate to parent skill. This is a strategic conflict, not a grilling problem.

### Type 3: Circular Dependencies
A depends on B and B depends on A.
- Response: Break the cycle by treating one side as an assumption. Document and plan to revisit.
- Escalation: If both sides resist assumption (both are high-stakes), escalate to brainstorming.

### Type 4: Value Judgment
The decision depends on subjective preference, not objective criteria.
- Response: "This is a value judgment. The options are [A] and [B]. The difference is [X]. Which do you prefer, and is this preference strong enough to justify the trade-off?"
